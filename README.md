# 介绍

Hei.Cnblog.Tools 是一个帮助你使用md编辑器写文章快速到上传图片到博客园，快速发文到博客园的工具；（当然也可以作为一个博客园图床工具但建议不要这么用比较博客园比较穷）

基本功能包括：

1. 拖拽图片.md文件上传图片到博客园；
2. 直接把本地.md博客投送到博客园发文页面，快速发文；
3. 记录几个常用博文目录；



本项目基于晓晨大佬的：[dotnet-cnblogs-tool](https://github.com/stulzq/dotnet-cnblogs-tool)
